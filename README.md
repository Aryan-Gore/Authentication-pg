# 🔐 Login & Signup Authentication Page

A modern and responsive Login & Signup (Authentication UI) page built using HTML, CSS, and JavaScript.  
This project features smooth UI transitions, animated panels, and a clean user-friendly design.

---

## 📌 About The Project

This project is a front-end authentication interface that includes:

- Login Form
- Registration Form
- Smooth sliding transitions between forms
- Modern UI design with animated elements
- Responsive layout

It is designed for integration with backend systems like Node.js, PHP, Django, or Spring Boot for full authentication functionality.

---

## ✨ Features

- 🔄 Toggle between Login & Register
- 🎨 Animated Background Shapes
- 💻 Responsive Layout
- 🧑‍💻 Clean Input Fields with Icons
- ⚡ Smooth Slide Animations
- 🔐 Password Field Support
- 📱 Mobile Friendly Design

## 🎯 How It Works

- The page contains two panels:
  - Login Panel
  - Register Panel
- JavaScript handles the toggle between login and signup forms.
- CSS animations provide smooth sliding transitions.

Currently, this project is front-end only.  
It can be connected to a backend server for full authentication functionality.
